# TMDB 클론 사이트 프로젝트

React를 이용한 TMDB 클론 사이트입니다.

### 주소

https://tmdb-made-by-kangwoo.netlify.app/

### 설치

**패키지 설치**

```bash
npm install
```

**env 추가하기**

FE 폴더 바로 아랫단에 `.env` 추가 후 `.env`파일 안에 다음과 같이 적고 저장

```
REACT_APP_API_CODE = 본인의 TMDB API CODE  //따옴표 없이 입력
```

**실행**

```bash
npm start
```

**빌드**

```bash
npm build
```
